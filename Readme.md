# NextHAM: Inference Pipeline

This repository is a lightweight inference branch for the paper **[[ICLR 2026] NextHAM: Advancing Universal Deep Learning for Electronic-Structure Hamiltonian Prediction of Materials]**.

Unlike the [main branch](https://github.com/DavidYin94/NextHAM) (which includes full training, testing, and ground-truth accuracy comparisons), this branch focuses strictly on the **inference pipeline**. It is heavily simplified, runs much faster, and is specifically designed for predicting the Hamiltonian of new material structures.

## ⚙️ Environment Requirements

The environment dependencies are identical to the main repository. Please refer to the [NextHAM Main Branch](https://github.com/DavidYin94/NextHAM) for detailed installation instructions.

## 🚀 Quick Start & Usage

We provide a comprehensive example script, `full_inference_pipeline.sh`, to demonstrate the complete workflow. 

### 0 Pre-requisite: Generate Zeroth-Step Hamiltonian
Before running the pipeline, you need to generate the zeroth-step Hamiltonian using the `get_hs` code.
- **Source Code**: [abacus-develop/largescale](https://github.com/goodchong/abacus-develop/tree/largescale)
- Compile and run this code on your target material samples. In our examples, we use an 8-atom Si system whose outputs are saved in `get_hs_res/8-atom/`.


### 🏃‍♂️ Pipeline Breakdown

The `full_inference_pipeline.sh` executes the following three main steps:

#### 1. Pre-processing
Run `pre_process.py` (located in `pre_post_process/`) to prepare your data.

```bash
python pre_process.py \
    --read-path "${BASE_DIR}/get_hs_res/8-atom/" \
    --save-path "${BASE_DIR}/data/" \
    --dataset-path "${BASE_DIR}/datasets/"
```

* `--read-path`: The directory containing the zeroth-step Hamiltonian generated by the `get_hs` code. Change this to your target sample's directory.
* `--save-path` & `--dataset-path`: Directories where the processed data will be temporarily stored for inference.

#### 2. Combine Data and Run Inference
Return to the base directory, combine the processed data, and execute the inference script.

```bash
python combine_data_infer.py
sh scripts/infer/infer.sh
```

#### 3. Post-processing
Run `post_process.py` (located in `pre_post_process/`) to finalize the results and generate plots.

```bash
python post_process.py \
    --prediction-path "${BASE_DIR}/data/input_inference_out.pth" \
    --stru-file "${BASE_DIR}/get_hs_res/8-atom/STRU" \
    --data-dir "${BASE_DIR}/get_hs_res/8-atom/OUT.ABACUS" \
    --save-path "res_8_split/plots/" \
    --fermi 2.1700249049
```

* `--prediction-path`: This **must** exactly match the output file generated by `infer.sh` (e.g., `input_inference_out.pth`).
* `--stru-file` & `--data-dir`: Paths pointing to your original structure and ABACUS output files.
* `--save-path`: Directory to save the final post-processed results and plots.
* `--fermi`: The Fermi energy of your specific system. Modify this value according to your material.
